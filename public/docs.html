GET /api/v1/many/:type

animedas.up.railway.app/api/v1/random/waifu

Returns an image list of 25 images according to the specified type

Params Description
type types: maid, waifu, smug, genshin
when using the "random" type the api will return random images

Response:

        {
    "message": "Successful",
    "content": [animes]
}

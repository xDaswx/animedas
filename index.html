Documentation

If you want to use the Animédas API without any problems, please refer to the documentation. The API is designed to provide access to the vast collection of anime-inspired icons available on our website.

Versatile

API contains images with tags and categories to make it easier for devs to use. Tags are keywords that describe the content of the icons, allowing users to easily find what they are looking for. Categories are a way of grouping icons by theme, such as anime characters.
